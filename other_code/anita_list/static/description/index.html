Anita list is a advance list for odoo, it has lots of features and it is very easy to use. if you need a evaluation version, please visit our website. https://odoo.funenc.com(click the website button) or you contact us via our email: odoo@funenc.com, The test account is admin/admin, do not change the password please.

(Current Release 14.0.0.2 -- 2022.05.31)

Fixed Header

Real Fixed Header, the header is fixed when scroll down.

Fixed Footer

Real Fixed Footer, the footer is always at the bottom of the page.

Fixed Left

Real fixed left, the left side is always at the left of the page.

Fixed Right

Real Right Side, the right side is always at the right of the page.

Groups

this plugin support x2many list!

Serials

You can config to show serials in the list or not.

Expand Row

You can config to show the expand_row, this is import for show the detail data in the list.

Search Row

You can config to show the expand_row, this is import for show the detail data in the list.

Time Range Select

it is a good choice to use a datetime picker to select the time range.

X2many Support

this plugin support x2many list!

Column Config

Column Config, you can config the column width mode, name, group, order, locked left and locked right, and visiblility. you also can show fields that not show in the list.

Settings

You can config to enable the list, show serials, search row, expand row template, it alse support tree grid mode.
